    - Empty array returns empty groups
    - Falls back to lastModified when dateTaken is undefined
    - Group has correct startTime and endTime
    - Threshold of 0ms groups only identical timestamps
    - Large threshold groups everything into one group

    Sorting tests (pure function):
    - Sort by filename uses natural sort (IMG_2 before IMG_10)
    - Sort by dateTaken (ascending and descending)
    - Sort by size (ascending and descending)
    - Sort by dimensions (megapixels = width * height, ascending and descending)
    - Images without dateTaken sort to end when sorting by date
    - Images without dimensions sort to end when sorting by dimensions
  </behavior>
